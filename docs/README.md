# 💧 Trevari - 물 섭취를 '의미 있는 변화'로 해석하는 습관형 앱

> 물을 더 많이 마시게 만드는 앱이 아니라, 물을 마신 '나의 상태'를 이해시켜주는 앱

## 📋 프로젝트 개요

Trevari는 바쁜 일상 속에서 물 섭취를 관리하고 싶지만, 기존 앱의 번거로운 기록 방식과 실패에 대한 부담을 느끼는 사용자들을 위한 습관형 앱입니다.

### 핵심 가치

- **무부담 기록**: ml 입력 없이 간단한 선택형 입력
- **실패 개념 제거**: 연속 기록(streak) 없음, 누락일도 괜찮음
- **의미 있는 해석**: AI가 물 섭취 패턴을 생활 변화와 컨디션으로 해석

## 🎯 주요 기능

### MVP (P0)
- 무부담 물 섭취 기록 (선택형 버튼)
- 실패 개념 제거 (연속 기록 없음)
- AI 기반 의미 해석 리포트 (Gemini API)

### 향후 기능 (P1, P2)
- 맥락 기반 알림 (Opt-in)
- 해석 보조 질문
- 주간 요약 공유

## 🛠 기술 스택

- **Frontend/Backend**: Next.js (App Router)
- **Database/Auth**: Supabase
- **AI**: Gemini API
